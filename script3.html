🎯 سكربت احترافي لإضافة VLAN على MikroTik بضغطة زر!

📌 من VLAN ID رقم 2 إلى 254 – بدون تعب ولا تكرار!

🔧 هل أنت مدير شبكة أو فني مايكروتك؟

وفر وقتك وجهدك مع أقوى سكربت أوتوماتيكي لإضافة جميع إعدادات الـ VLAN دفعة واحدة من التيرمنال!

💡 ما هو الـ VLAN؟

شبكة VLAN تسمح بفصل الشبكات على نفس الواجهة الفيزيائية، وتمنحك مرونة وتحكم أكبر في إدارة الأجهزة والمستخدمين.

⚙️ ماذا يفعل السكربت؟
  • ✅ يضيف VLAN IDs من 2 إلى 254 بشكل تلقائي
  • ✅ يجهّز كل إعدادات الهوتسبوت المطلوبة لكل VLAN:
    • IP Address
    • IP Pool
    • DHCP Server
    • DHCP Network
    • NAT
    • Hotspot
🚀 المميزات:
  • 🧠 سهل الاستخدام – لا يحتاج خبرة كبيرة
  • ⏱️ يوفر ساعات من العمل اليدوي
  • 📡 مثالي لمقدمي خدمة الإنترنت (ISP) أو الفنادق أو المدارس والشبكات الكبيرة
📝 قبل الاستخدام:
  1. عدل الـ VLAN ID إذا احتجت أقل أو أكثر
  2. تأكد أن الرنج 172.18.x.x غير مستخدم
  3. غيّر اسم البروفايل المستخدم في الهوتسبوت
  4. عدل منفذ الخروج (Bridge1) حسب شبكتك
📜 السكربت:

🔹 السكربت الأول (لإضافة VLANs):

:for num2 from=2 to=254 do={
:delay 1s
/interface vlan add name="AP-$num2" comment="M.Majd" interface=OUT vlan-id="$num2"
}

🔹 السكربت الثاني (لضبط كل الإعدادات):

:foreach i in=[/interface vlan find comment="M.Majd" ] do={ 
:local a [/interface vlan get $i vlan-id];
:local b [/interface vlan get $i name];
:delay 1s
/ip address add address="10.$a.0.1/24" disabled=no interface="$b";
/ip pool add name="$a" range="10.$a.0.3-10.$a.0.254" ;
/ip dhcp-server network add address="10.$a.0.0/24" gateway="10.$a.0.1" dns-server="10.$a.0.1" ;
/ip firewall nat add action=masquerade chain=srcnat src-address="10.$a.0.0/24" ;
/ip dhcp-server add name=$a disabled=no address-pool=$a interface=$b lease-time="3h"; 
/ip hotspot add name=$b address-pool=$a interface=$b disabled=no profile=hsprof1 idle-timeout=3m addresses-per-mac=unlimited ;
}

💡 بعد التنفيذ، ستصبح جميع VLANs معدّة وجاهزة للعمل فورًا.
