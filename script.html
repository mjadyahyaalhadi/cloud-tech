حل عملي لمشكلة نقل يوزرات الهوتسبوت مع الوقت والرصيد المتبقي
----- MikroTik Hotspot Users Transfer

كثير من المهندسين والمشبكين عندهم مشكلة لما يعملوا Export/Import لليوزرات حق الهوتسبوت وليس اليوزر مانجر من راوتر الى روتر آخر - تروح كل الاستخدامات السابقة ويتصفروا الكروت ويرجع اليوزر بوقت ورصيد جديد وكأنه ما استخدم شيء!

هنا يجي دور هذا السيكربت:
  • ✔️ يحسب الوقت المتبقي لكل يوزر.
  • ✔️ يحسب الرصيد المتبقي (Download/Upload).
  • ✔️ يعدّل القيم لكل يوزر تلقائيًا.
  • ✔️ يوقف الحسابات المنتهية (Disabled) او يحذفها حسب طلبك.

وفي النهاية يعمل لك نسخة جاهزة باسم: Eng.AlmamooN.Cards.rsc فيها كروت الهوتسبوت مع الميجا المستخدم والوقت المتبقي، ويمكن استيرادها في أي راوتر آخر.

مثال:
الوقت المتبقي = 03:00:00
الميجا المتبقي = 3200 ميجا
اذا استخدم الزبون 1000 ميجا وساعة كاملة
الكرت في النسخة الاحتياطية يصبح:
الوقت المتبقي = 02:00:00
الميجا المتبقي = 2200 ميجا
طريقة الاستخدام:
  • وقت فقط: :global MODE "time";
  • تحميل فقط: :global MODE "bytes";
  • وقت + تحميل (افتراضي): :global MODE "both";
  • حذف الكروت المنتهية: :global CLEAN_EXPIRED true;
  • تعطيل الكروت المنتهية: :global CLEAN_EXPIRED false;
  • اسم ملف النسخة الاحتياطية الافتراضي: Eng.AlmamooN.Cards.rsc ويمكن تغييره بتعديل EXPORTFILE

تنبيه: لا تنفذ السيكربت قبل عمل نسخة احتياطية:

/system backup save name=BeforeBackupHcards
بداية السيكربت:
:global MODE "both";:global CLEANEXPIRED true;:global EXPORTFILE "Eng.AlmamooN.Cards";:log info ("[Hotspot-Migrate] Start | MODE=" . $MODE . " | CLEANEXPIRED=" . $CLEANEXPIRED . " | FILE=" . $EXPORTFILE);:foreach i in=[/ip hotspot user find] do={:local uname [/ip hotspot user get $i name];:local limitupt [/ip hotspot user get $i limit-uptime];:local uptime [/ip hotspot user get $i uptime];:local limitbytes [/ip hotspot user get $i limit-bytes-total];:local bin [/ip hotspot user get $i bytes-in];:local bout [/ip hotspot user get $i bytes-out];:local usedbytes ($bin + $bout);:if ($CLEANEXPIRED = true) do={:local expired false;:if (($limitupt > 0s) && ($uptime >= $limitupt)) do={:set expired true};:if (($limitbytes > 0) && ($usedbytes >= $limitbytes)) do={:set expired true};:if ($expired = true) do={:log info ("[Hotspot-Migrate] REMOVE expired user: " . $uname);/ip hotspot user remove $i;:continue}};:if (($MODE = "time") || ($MODE = "both")) do={:if ($limitupt > 0s) do={:if ($uptime < $limitupt) do={:local newUpt ($limitupt - $uptime);/ip hotspot user set $i limit-uptime=$newUpt} else={/ip hotspot user set $i disabled=yes comment="end uptime"}}};:if (($MODE = "bytes") || ($MODE = "both")) do={:if ($limitbytes > 0) do={:if ($usedbytes < $limitbytes) do={:local newBytes ($limitbytes - $usedbytes);/ip hotspot user set $i limit-bytes-total=$newBytes} else={/ip hotspot user set $i disabled=yes comment="end download"}}}};/ip hotspot user export file=$EXPORTFILE;:log info ("[Hotspot-Migrate] DONE | Exported file: " . $EXPORTFILE . ".rsc");
بعد النسخة الاحتياطية:

انسخ الملف Eng.AlmamooN.Cards.rsc للراوتر الجديد ونفذ: 

/import file=Eng.AlmamooN.Cards.rsc

وسيتم إضافة الكروت مع الميجا والوقت المتبقي بالكامل.
